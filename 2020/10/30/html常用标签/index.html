# html 常用标签

# 格式化标签

格式标签：

		<br />  换行
		<p>   </p>  换段
		<hr />   水平线分割
		列表：
		<ul>   </ul>  无序列表
		<ol>   </ol>  有序列表，type类型值：A a I i 1 start属性表示起始值
		<li>   </li>  列表项
		      <dl>   </dl>   自定义列表
			  <dt>   </dt>   自定义列表头
			  <dd>   </dd>   自定义列表内容
	    <div>   </div>  常用于组合块级元素

# 图像标签

图像标签：

<img src="#" alt="加载失败" title="这是一张图片" width="200"
		height="200" border="0"/>
		src: 图片路径
		alt: 加载失败信息
		title: 文字提示属性
		width: 宽度
		height: 高度
		border: 边框线粗细

# 超链接标签

超链接标签:

		<a href="#" target="_blank"></a>
		href: 链接跳转地址
		target: 链接打开方式
			_blank 新窗口
			_parent 父窗口
			_self 本窗口（默认）
			_top 顶级窗口
			framename 窗口名
			title: 文字提示属性
		锚点链接：
			定义：
			<a id="ab"></a> 旧方法: <a name="ab"></a>
			使用:
			<a href="#ab"></a>

# 表格标签

表格标签：

<table cellpadding="单元格边距" cellspacing="单元格间距">
			<caption>标题</caption>
			<th colspan="2">表头,占据两列单元格</th>
			<th rowspan="2">表头,据两行单元格</th>
			<td>表格内容</td>
</table>

# 表单标签

表单标签：

<form action="#" method="get/post">
			action:提交的目标地址
			method: 提交方式
				get: URL地址可见，长度限制，不安全
				post: URL地址不可见，安全
			<input type="text" name="" />
			input定义输入字段。
				type属性:
					text: 文本
					password: 密码
					checkbox: 多选
					radio: 单选
					file: 文件上传
					button: 普通按钮
					submit: 提交按钮
					image: 提交图片
					...
			<select name="" size="3" disabled="false" multiple="multiple">
				<option value="1" selected="selected"> </option>
				value: 下拉属性的值
				sekected: 默认下拉指定项
				<textarea>多行文本输入</textarea>
				name: 定义名称，存储文本域中的值
				cols: 可见列数
				rows: 可见行数
				diabled: 是否禁用
				readonly: 只读
			</select>
			name: 定义名称，存储下拉值
			size: 菜单项目的数目 
			disabled: 禁用菜单
			multiple: 多选
</form>

# 框架标签

框架标签：

<iframe src="" name="" width="" height=""></iframe>

# 多媒体标签

音频标签：

<audio src="" >
			当前浏览器不支持audio
		</audio>

视频标签：

	<video width="800" height="">
		<source src="myvideo.mp4" type="video/mp4"></source>
		<source src="myvideo.ogv" type="video/ogg"></source>
		<source src="myvideo.webm" type="video/webm"></source>
		<object width="" height="" type="application/x-shockwave-flash" data="myvideo.swf">
			<param name="movie" value="myvideo.swf" />
			<param name="flashvars" value="autostart=true&amp;file=myvideo.swf" />
		</object>
		当前浏览器不支持 video直接播放，点击这里下载视频： <a href="myvideo.webm">下载视频</a>
	</video>
